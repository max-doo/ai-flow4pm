## 项目概述

AI Flow 产品设计系统是一个面向产品经理的智能化设计工具，旨在利用 AI 技术简化产品设计流程。

## 核心功能

### 🎨 智能原型生成

通过自然语言描述，AI 可以快速生成产品原型：

- 支持多种设计风格 
- 自动适配移动端和桌面端
- 一键导出设计稿

### 🤖 AI 驱动的用户研究

利用大语言模型分析用户反馈：

- 自动提取用户痛点
- 生成用户画像
- 预测用户行为趋势

### 📊 数据可视化

将复杂数据转化为直观图表：

- 支持多种图表类型
- 实时数据更新
- 交互式探索

## 技术栈

| 技术 | 用途 |
|------|------|
| Vue 3 | 前端框架 |
| TypeScript | 类型安全 |
| Tailwind CSS | 样式系统 |
| OpenAI API | AI 能力 |
